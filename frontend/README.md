# Frontend

## Setup Environment

- Install [NodeJS v17.2.0](https://nodejs.org/en/blog/release/v17.2.0/)

Now in the project folder run:
```sh
npm install # install node packages
npm start # start the aplication
```

## Run Build

```sh
npm build
```

## Run Tests

```sh
npm test
```
